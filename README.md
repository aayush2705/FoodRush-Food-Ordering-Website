# 🍔 FoodRush - Food Ordering Website

A modern and responsive **Food Ordering Web Application** built using **HTML, CSS, Bootstrap, and JavaScript**, inspired by platforms like Swiggy and Zomato.

---

## 📌 Features

✔️ Select city and restaurants dynamically

✔️ View restaurant menu with images

✔️ Shuffle menu for each restaurant

✔️ Add to cart (up to 10 items per product)

✔️ Cart system with total calculation

✔️ Checkout with delivery details

✔️ UPI QR payment system

✔️ Order success confirmation page

✔️ Login & Signup using LocalStorage

✔️ Fully responsive Bootstrap UI
